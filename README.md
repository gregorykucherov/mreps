## Installation
```
git clone https://github.com/gregorykucherov/mreps.git
cd mreps
make
```
## Getting started
```
./mreps -allowsmall -s ATCACTAGaagaACAGAACAATTACTTaatag
./mreps -res 1 -fasta test/sequence.fasta
```
